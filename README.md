# Task Tracker Web App

A simple full-stack task management web application built with Python, Flask, and SQLite.

## Features

- Add tasks through a web interface
- Store tasks in a SQLite database
- Display saved tasks dynamically
- Delete tasks from the list

## Technologies Used

- Python
- Flask
- SQLite
- HTML/CSS

## How It Works

Users can enter tasks through a form on the web page.  
The backend processes the request and stores the task in a SQLite database.  
Tasks are then retrieved and displayed dynamically on the page.
